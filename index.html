Project Overview

For the course project, the following was required:

The creation of a Shiny application
1. Hosted on the RStudio Shiny server (https://sgardiner.shinyapps.io/ShinyProject)
2. Some sort of input (my app used a slider widget, textboxes and a Submit button)
3. Reactive output (my app updated a bar plot based on the slider widget)
4. Documentation, so that a novice user could use the app (while the app is very straightforward, instructions were posted in the main panel below the plot)

A reproducible pitch presentation
1. Created using RStudio Presenter hosted on RPubs (http://rpubs.com/SGardiner)
2. Including embedded R code that gets run when the slide deck is created (slides 3 and 4 of this presentation)
3. Must be 5 pages, including the title slide

The original data used for this dataset was from the Rec.Sport.Soccer Statistics Foundation website - http://www.rsssf.com/tablese/eng2016.html.
It showed the game outcomes for the English Premier League 2015-2016 season, historic because the league winner, Leicester City, was considered the greatest longshot in English soccer history.

Prediction algorithm
1. The standings.csv dataset was used as a starting point
2. Recent performance was added as a predictor by taking the mean of the outcomes of the previous 5 games
3. A machine learning approach was used for the prediction algorithm, using a Random Forests model, with k-folds = 5 for cross validation.

The slider widget (Week) and text Select boxes (Home and Away) are the inputs to the prediction function. The prediction simply shows the game winner (Home/Away/Draw).
